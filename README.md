# Dee Keyboard / Qwen Keyboard

Android voice keyboard / IME research project for local and PC-assisted ASR, Chinese input, punctuation, handwriting, and flow/swipe typing experiments.

## Current build line

Latest local development build at time of initial GitHub push:

- `v4.5.0-jiufang-pad-sucheng`
- Package: `ai.qwenkeyboard.voicekeyboard`
- Main app module: `QwenVoiceBenchmark/app`

## Highlights

- Android IME keyboard UI
- Local ASR model integrations via sherpa-onnx experiments
- PC ASR server bridge scripts
- Pinyin Chinese input
- 九方-style 3x3 pad prototype
- 速成 starter mode
- Offline / PC AI punctuation selection
- UI sizing controls
- Flow/swipe typing prototype

## Notes

Generated APKs, model files, logs, OpenClaw runtime state, memory files, and token/config files are intentionally ignored from Git.
